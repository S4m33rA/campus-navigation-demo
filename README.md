# Campus Compass (Contest MVP)

UX-focused mobile navigation concept app for university judges.

## What this demo includes

- One-tap "Get Me to Class" flow
- Route preference selection (Fastest, Step-free, Safer at night)
- Live turn-by-turn style guidance card
- Arrival confirmation + 1-tap feedback capture
- Accessibility toggles: high contrast, larger text, accessibility mode

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start Expo:

   ```bash
   npm run start
   ```

3. Open on device or simulator from the Expo QR screen.

## Open from desktop

You can run this app directly in your desktop browser:

```bash
npm run desktop
```

Then press `w` in the Expo terminal (if it does not open automatically) and use the local web URL in Chrome, Safari, or Edge.

## Pitch angle for judges

This app is designed to reduce student wayfinding stress by combining timetable context, accessibility-aware routing, and simplified step-by-step guidance in one mobile experience.
